🚀 Setup Instructions

  1. Download the setup script by clicking the button above
  2. Open Terminal and navigate to where you want the project
  3. Make script executable: chmod +x create-project.sh
  4. Run the script: ./create-project.sh
  5. Open in VS Code: code cybercrime-assistance-system
  6. Install dependencies:
    cd backend && npm install
    cd ../frontend && npm install
  7. Configure environment: Copy .env.example to .env in both folders
  8. Start the application:
    # Terminal 1
    cd backend && npm run dev

    # Terminal 2
    cd frontend && npm start
⚠️ Important: Make sure you have Node.js (v14+) and MongoDB installed before running the project.
