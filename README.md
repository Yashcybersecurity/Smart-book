#  Smart Book (Farexo)

A modern, full-stack ride-sharing application built with Next.js, Firebase, and Google Maps integration. Smart Book provides intelligent location suggestions, real-time mapping, and seamless authentication.

## ✨ Features

- **Google Maps Integration** - Places Autocomplete for pickup and destination locations
- **Authentication** - Secure user authentication with NextAuth
- **Firebase Backend** - Real-time database and cloud storage
- **Mobile Support** - Capacitor integration for iOS/Android deployment
- **Next.js 14** - Modern React framework with optimized performance
- **India-Specific** - Geo-restricted location services for Indian cities

## 🛠️ Tech Stack

### Frontend
- **Next.js 14.2.3** - React framework for production
- **React 18.3.1** - UI library
- **TailwindCSS** - Styling (configured in project)

### Backend & Services
- **Firebase 12.9.0** - Real-time database and authentication
- **Firebase Admin SDK 13.7.0** - Server-side Firebase operations
- **NextAuth 4.24.13** - Authentication and session management

### Mobile
- **Capacitor 8.1.0** - Build native iOS/Android apps with web technologies

### Development
- **Playwright Core** - E2E testing framework

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- **Node.js** (v16 or higher)
- **npm** or **yarn**
- **Git**
- Google Cloud Account (for Maps API)
- Firebase Project
